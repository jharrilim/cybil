# `c y b [ i ] l .`

Cybil is a Feed-Forward Neural Network written in Rust. This project aims to be used with WASM, and will provide WASM bindings for usage in Javascript/browsers.

## Development

### Getting Started

First enter the core dir

```sh
cd ./lib/core
```

### Build

```sh
cargo build
```

### Test

```sh
cargo test
```
